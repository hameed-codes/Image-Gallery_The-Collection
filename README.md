## Features

- **Masonry‑like grid** — images arrange naturally in columns, adapting to any screen size.
- **Responsive design** — 4 columns on large screens, 3 on tablets, 2 on mobile.
- **Click to enlarge** — each image opens in a full‑screen lightbox with a blurred backdrop.
- **Smooth hover effect** — subtle scale and shadow when hovering over images.
- **No dependencies** — vanilla HTML/CSS/JS, works out of the box.

---

## Technologies Used

- **HTML5** : semantic structure
- **CSS3** : custom properties, `clamp()` for fluid typography, CSS Grid (via `columns`)
- **JavaScript (ES6)** : event listeners, DOM manipulation

---


## Setup & Usage

1. **Clone or download** the repository.
2. Open `index.html` in any modern web browser.
3. Click on any image to view it in the lightbox overlay.
4. Click anywhere on the overlay to close it.

No build steps, no server required — it’s ready to use.
